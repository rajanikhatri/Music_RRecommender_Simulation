# Reflection

## Biggest Learning Moment

The biggest learning moment for me was seeing how a recommender can feel smart even when the math is very simple. A few clear rules for genre, mood, energy, and tempo were enough to create recommendations that mostly made sense.

## How AI Tools Helped

AI tools helped me move faster when planning the file structure, organizing the scoring logic, and rewriting documentation in a cleaner way. They were especially useful for turning rough ideas into code that was easier to read.

## Where AI Suggestions Needed Double-Checking

I still had to double-check the AI suggestions against the actual CSV headers and the starter tests. Some ideas sound good in theory, but they only work if they match the real data and the existing project structure.

## What Was Surprising

One surprising thing was how strongly the scoring weights affect the results. A simple bonus like `+2.0` for genre match can change the ranking a lot, even when another song is closer in energy or tempo.

## What I Would Improve Next

If I kept working on this, I would add a larger dataset, more user preference options, and a better balance between similarity and variety. I would also like to compare this simple content-based approach with a history-based recommender.
